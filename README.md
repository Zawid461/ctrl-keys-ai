# 📖 Overview

CTRL KEYS transforms a simple project idea into a complete software application through a coordinated team of AI agents.

Unlike traditional AI coding assistants, CTRL KEYS follows a structured software engineering workflow inspired by real development teams. Each agent has a dedicated responsibility, enabling modular reasoning, cleaner project generation, and automated debugging.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Zawid461/ctrl-keys-ai.git
```

Move into the project

```bash
cd ctrl-keys-ai
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

Windows

```bash
venv\Scripts\activate
```

Linux/macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=your_key
TAVILY_API_KEY=your_key
```

Run CTRL KEYS

```bash
streamlit run ui.py
```

# 🚀 How It Works

1. Enter a project idea.
2. Planner Agent creates a development plan.
3. Architect Agent designs the project structure.
4. Coder Agent generates production-ready code.
5. Reviewer validates generated code.
6. File Writer creates the project.
7. Executor runs the project automatically.
8. Debugger fixes runtime errors if detected.
