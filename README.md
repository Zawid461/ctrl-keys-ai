# 🚀 CTRL KEYS

> AI-Powered Autonomous Software Engineering Platform

CTRL KEYS is a multi-agent AI software engineering platform that researches, plans, generates, reviews, executes, and debugs complete software projects using Large Language Models and live documentation.

---

## ✨ Features

- 🧠 AI Project Planning
- 🏗 Software Architecture Generation
- 💻 Automatic Code Generation
- 📂 Multi-file Project Creation
- ▶ Code Execution
- 🐞 AI Debugging
- 🌐 Live Documentation Search (Tavily)
- 📜 Real-time Activity Logs
- 📊 Streamlit Dashboard
- 📁 Project History
- 📦 ZIP Download
